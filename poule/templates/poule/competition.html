Competition Maker

Select teams, days and times to generate games. Each team will face each other team once or twice, depending on whether you select Return Games. The teams for each game are picked with the Round Robin algorithm. The dates for the games are determined by the start date, game days and games per day. Starting with the start date each selected game day will get the number of games per day on the selected times.
